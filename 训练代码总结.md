# 训练代码总结

这个文件汇总了本项目中我实际使用过的训练代码、各脚本作用、核心流程和运行方式。

## 1) `baseline_model.py`（初版 baseline）

### 作用
- 读取训练集 `student_train_full.csv` 与测试集 `public_test_questions.csv`
- 清洗 `price`（去 `$`、`,`、转 float、删除空标签）
- 删除全空列和明显无关列
- 用 10 个手工选择的数值特征训练 `RandomForestRegressor`
- 输出验证集 `MAE`、`R^2`，并生成预测文件 `public_test_predictions.csv`

### 核心特征
- `accommodates`
- `bathrooms`
- `bedrooms`
- `beds`
- `latitude`
- `longitude`
- `review_scores_rating`
- `availability_365`
- `number_of_reviews`
- `calculated_host_listings_count`

### 运行命令
```bash
python baseline_model.py
```

---

## 2) `improved_baseline.py`（提分 baseline，主力版本）

### 作用
- 在初版清洗基础上，进一步删除无帮助列
- 自动对齐 train/test 特征，只保留数值特征
- 删除训练集中“全空列”
- 使用中位数填充缺失值
- 使用 `ExtraTreesRegressor` + `log1p(price)` 训练
- 预测时 `expm1` 还原价格
- 生成 `public_test_predictions_improved.csv`

### 提分关键
- 对目标 `price` 做对数变换（`np.log1p`）缓解长尾分布
- 使用 `ExtraTreesRegressor` 提升非线性拟合能力

### 运行命令
```bash
python improved_baseline.py
```

---

## 3) `focused_feature_model.py`（指定特征定向训练）

### 作用
- 仅使用你指定的重点特征训练：
  - `property_type`
  - `bathrooms`
  - `latitude`
  - `longitude`
  - `host_since`（转换为 `host_tenure_days`）
  - `number_of_reviews`
  - `review_scores_value`
- 类别特征 `property_type`：众数填充 + OneHot 编码
- 数值特征：中位数填充
- 模型：`RandomForestRegressor`（同样使用 `log1p` 训练）
- 输出文件：`cys_day2_focus_preds.csv`（含 `id,predicted_price`）

### 运行命令
```bash
python focused_feature_model.py
```

---

## 5) 训练流程总览（统一）

1. 读取数据  
2. 清洗 `price`（仅训练集）  
3. 特征工程 / 特征选择  
4. 训练集-验证集划分（`train_test_split`, `random_state=42`）  
5. 缺失值填充  
6. 训练回归模型  
7. 计算 `MAE` 与 `R^2`  
8. 在 `public_test_questions.csv` 上预测并导出 csv
